# المطلوب من ChatGPT

## 1. حلل الصورة المرفقة بدقة

هذه صورة للوحة تحكم الأدمن (Admin Dashboard) بعد التطوير لمشروع Next.js (سحابة الأمن السيبراني).

## 2. أخرج تقريراً بالعربية يحتوي على:

### أ. الهيكل العام
- ما هي الأقسام / الكارتات / المكونات الموجودة في الصورة؟
- كيف موزعة على الصفحة (يمين/يسار/وسط/أعلى/أسفل)؟

### ب. كل عنصر بالتفصيل
لكل عنصر (مثل كارت إحصائي، جدول، رسم بياني، زر، شارت، إلخ):
- **المحتوى**: ماذا يعرض؟ (أرقام، نصوص، روابط)
- **النمط البصري**: ألوان، ظلال، أيقونات، حواف
- **الإجراءات (Actions)**: هل هو قابل للنقر؟ هل يفتح صفحة أخرى؟
- **البيانات**: هل هي إحصائيات افتراضية أم حقيقية؟

### ج. التصميم العام
- Color scheme (ألوان الخلفية، النصوص، الأزرار)
- Typography (حجم الخطوط، نوعها)
- Spacing (هوامش، فراغات)
- Responsive behaviour (كيف سيبدو على الجوال؟)

### د. الميزات الجديدة
- ما الذي يختلف عن لوحة تحكم تقليدية؟
- هل في عناصر تفاعلية (مخططات متحركة، أزرار تبديل، قوائم منسدلة)؟
- هل في أي عنصر ملاحظ (مثل إشعارات، تنبيهات، تقدم)?

### هـ. التوصيات التقنية
- هل المكونات قابلة لإعادة الاستخدام؟
- ما مكتبات UI التي توحي بها الصورة؟ (Tailwind, Material UI, Shadcn, إلخ)
- هل البيانات ديناميكية أم static؟

## ملاحظات مهمة للمطور
- المشروع: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- قاعدة البيانات: PostgreSQL مع Prisma
- المصادقة: JWT (مخصص) مع أحرف عربية (RTL)
- الصفحة ستكون في: `app/(dashboard)/admin/page.tsx`
- الكود الموجود حالياً: لوحة تحكم أساسية بإحصائيات وجداول
